This isn't real!
=========  
Find out who is the singer of the song :)  
  
Requiriments.

  - ```git``` (of course we need git)
  - ```make``` (install via ```apt-get``` or ```brew```)
  - ```bundler``` (```gem install bundler```)

Installation
--------------

```sh
make install
```
Developement
--------------
```sh
make dev
```
